CSS LEARNING OVERVIEW 

1. CSS OVERFLOW PROPERTIES (03_Overflow.html_Ellipsis.html)
   - overflow: visible | hidden | scroll | auto
     * visible (default): Content overflows container
     * hidden: Content clipped, no scrollbars
     * scroll: Always shows scrollbars
     * auto: Shows scrollbars only when needed
   
   - overflow-x & overflow-y: Control specific axes
   
   - text-overflow: ellipsis | clip
     * ellipsis: Shows "..." for overflowing text
     * clip: Cuts off text abruptly
   
   - white-space: nowrap - Prevents text wrapping

2. CSS TRANSFORM PROPERTIES (04_CSS_Transform.html)
   - transform: Changes element appearance without affecting layout
   - Common transform functions:
     * scale(x, y) - Resizes element
     * rotate(deg) - Rotates element (positive = clockwise)
     * translate(x, y) - Moves element
     * translateX() / translateY() - Move along single axis
   
   - Transform chaining: Multiple transforms in one declaration
     Example: transform: scale(0.5) rotate(-45deg) translate(100px);
   
   - transform-origin: Sets pivot point for transformations

3. CSS TRANSITIONS (05_CSS-Animation-Transition.html)
   - transition: property duration timing-function delay
     Example: transition: transform 1s linear 0.5s
   
   - What transitions can animate:
     * Transform properties
     * Colors (background-color, color, border-color)
     * Dimensions (width, height)
     * Position (left, top)
   
   - Applied on base state, triggered on state change (like :hover)

4. CSS ANIMATIONS (06_cricket-animation.html, 07_bounce-effect.html)
   - @keyframes rule: Defines animation sequence
     Syntax: @keyframes animation-name { 0% {...} 50% {...} 100% {...} }
   
   - animation property: shorthand for all animation settings
     Syntax: animation: name duration timing-function delay iteration-count direction
   
   - Animation properties:
     * animation-name: References @keyframes
     * animation-duration: How long animation lasts
     * animation-timing-function: Speed curve (ease, linear, etc.)
     * animation-delay: Delay before starting
     * animation-iteration-count: infinite | number
     * animation-direction: normal | reverse | alternate | alternate-reverse
   
   - Position requirements: left/top need position: relative/absolute/fixed

5. CSS CUSTOM PROPERTIES (VARIABLES) (08_CSS-Custom-property.html)
   - Defined in :root (global) or specific selectors (local)
     Example: :root { --primary-color: steelblue; }
   
   - Accessed with var() function
     Example: color: var(--primary-color);
   
   - Benefits: Reusability, maintainability, theme switching
   
   - calc() function: Perform calculations in CSS
     Example: width: calc(100% - 200px);
