CSS Flexbox & Grid Layout

1. Flexbox Recap (01_flex-recap.html)
   - Flex container: `display: flex`
   - Spacing: `gap` property
   - Responsive behavior: `flex-direction: column` on small screens
   - Wrapping: `flex-wrap: wrap`
   - Justify content: `justify-content: space-between`
   - Media queries for mobile responsiveness

2. CSS Grid Basics
   - Grid container: `display: grid`
   - Grid template columns: `grid-template-columns: repeat(6, 1fr)`
   - Grid template rows: `grid-template-rows: repeat(6, 200px)`
   - Image fitting: `width: 100%; height: 100%`

3. Complex Grid Layout (03_design-grid-gallary.html)
   - Grid span: `grid-column: span X` and `grid-row: span Y`
   - Layout classes: big-layout, normal-layout, wide-layout, small-layout
   - Responsive grid: Changing grid structure with media queries
   - Mobile adaptation: Switch to 2-column layout or block display

4. Grid Advanced Features (04_recap-grid.html)
   - Grid gap: `gap: 20px`
   - Multi-row spanning: `grid-row: span 5`
   - Multi-column spanning: `grid-column: span 3`
   - Fixed and flexible row heights

5. Grid Area Layout (05_grid-area.html, 06_responsive-grid.html)
   - Grid template areas for semantic layout
   - Naming areas: `grid-area: header`, etc.
   - Template structure: `grid-template-areas`
   - Responsive redesign with media queries:
     * Mobile: Single column stack
     * Tablet: Modified area arrangement
     * Desktop: Full grid layout

6. Key Responsive Techniques
   - Media queries: `@media screen and (max-width: 576px)`
   - Breakpoints: 576px (mobile), 577-992px (tablet)
   - Mobile-first vs desktop-first approaches
