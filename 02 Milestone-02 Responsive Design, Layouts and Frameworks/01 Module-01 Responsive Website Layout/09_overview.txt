### CSS Layout & Responsive Design - Learning Overview

### 1. **CSS Measurement Units**
- **Absolute Units (Fixed):**
  - `px` - Pixels (1px = 1 screen dot)
  - `pt` - Points (mainly for print)
  - `in`, `cm`, `mm` - Physical units
  
- **Relative Units (Flexible):**
  - `%` - Percentage of parent element
  - `em` - Relative to parent's font-size
  - `rem` - Relative to root (html) font-size
  - `vw/vh` - Viewport width/height percentage
  - `vmin/vmax` - Viewport minimum/maximum dimension
  - `ch` - Width of "0" character

### 2. **Flexbox Layout**
- **Container Properties:**
  - `display: flex` - Creates flex container
  - `flex-direction` - Row/column direction
  - `justify-content` - Horizontal alignment
  - `align-items` - Vertical alignment
  - `flex-wrap` - Wrapping behavior
  
- **Practical Application:**
  - Navigation menus (space-between)
  - Centering elements easily
  - Responsive form layouts

### 3. **CSS Grid Layout**
- **2D Layout System:**
  - `display: grid` - Creates grid container
  - `grid-template-columns` - Defines columns
  - `grid-template-rows` - Defines rows
  - `gap` - Spacing between grid items
  - `repeat()` function for repeating patterns
  - `fr` unit for fractional space
  
- **Real-world Examples:**
  - Calendar layout (7 columns)
  - Product grids
  - Responsive card layouts

  - Mobile-first approach (start small, add for larger screens)
  - Common breakpoints: 576px, 768px, 992px, 1200px
  - Tailwind breakpoints: sm, md, lg, xl, 2xl

### 5. **Responsive Design Techniques**
- **Key Principles:**
1. Viewport meta tag: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
2. Fluid images: `max-width: 100%; height: auto;`
3. Container pattern with max-width
4. Flexible units (%, rem, vw/vh)
5. Responsive typography with `clamp()`

- **Layout Patterns:**
- Flexible Grid: `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))`
- Flex wrapping: `flex-wrap: wrap`
- Aspect ratio containers
