## WHAT I LEARNED TODAY IN CSS:

### 1. THREE WAYS TO APPLY CSS:
1. INLINE CSS: style attribute directly in HTML tags
   Example: <p style="color: red;">Text</p>
   
2. INTERNAL CSS: <style> tags in HTML head section
   Example: <style>p {color: blue;}</style>
   
3. EXTERNAL CSS: Separate .css file linked with <link>
   Example: <link rel="stylesheet" href="style.css">

### 2. CSS SELECTORS (HOW TO TARGET ELEMENTS):
- ELEMENT SELECTOR: Targets HTML tags
  Example: p {color: blue;}
  
- CLASS SELECTOR: Targets elements with specific class (.)
  Example: .highlight {background: yellow;}
  Usage: <p class="highlight">Text</p>
  
- ID SELECTOR: Targets single element with specific id (#)
  Example: #header {background: gray;}
  Usage: <div id="header">Content</div>
  
- UNIVERSAL SELECTOR: Targets all elements (*)
  Example: * {margin: 0; padding: 0;}
  
- COMBINED SELECTORS: Multiple classes on one element
  Example: <p class="bold red big">Text</p>
  CSS: .bold {font-weight: bold;} .red {color: red;}
  
- GROUPING SELECTORS: Apply same styles to multiple selectors
  Example: h1, h2, h3 {color: blue;}
  
- ATTRIBUTE SELECTOR: Targets elements with specific attributes
  Example: input[type="password"] {color: pink;}

### 3. BOX MODEL (FUNDAMENTAL CONCEPT):
Every element is a rectangular box with 4 parts:
1. CONTENT: Actual text/images
2. PADDING: Space between content and border
3. BORDER: Line around padding
4. MARGIN: Space outside border

Example Box Model Implementation:
.box {
    width: 300px;          /* Content width */
    height: 200px;         /* Content height */
    padding: 20px;         /* Inner space */
    border: 2px solid black; /* Border line */
    margin: 30px;          /* Outer space */
}

### 4. BORDER PROPERTIES:
- border: width style color; /* 2px solid black; */
- border-radius: Makes rounded corners
- Individual sides: border-top, border-right, etc.

Border Examples:
.square {border: 2px solid black;}
.rounded {border: 2px dashed red; border-radius: 10px;}
.asymmetric {border-top: 5px solid blue; border-bottom: 5px dotted green;}

### 5. MARGIN & PADDING SHORTHANDS:
Four-value syntax: top right bottom left
Three-value: top right-left bottom
Two-value: top-bottom right-left
One-value: all sides

Examples:
margin: 10px 20px 30px 40px;  /* T R B L */
padding: 20px 40px;           /* TB RL */
margin: 15px;                /* All sides */

### 6. DISPLAY PROPERTIES:
- block: Takes full width, starts new line (div, p, h1-h6)
- inline: Takes only needed width, no new line (span, a, strong)
- inline-block: Mix of both - inline flow with block properties
- none: Completely removes element from layout
- visibility: hidden; hides element but keeps space

Display Examples:
.inline-example {display: inline; width: 100px;} /* Width ignored */
.inline-block-ex {display: inline-block; width: 100px;} /* Width works */
.hidden {display: none;} /* Element removed */
.invisible {visibility: hidden;} /* Space kept */

### 7. BACKGROUND PROPERTIES:
- background-color: Solid color background
- background-image: Image as background
- background-repeat: no-repeat, repeat-x, repeat-y
- background-position: left, center, right, top, bottom
- background-size: cover, contain, specific dimensions

Background Examples:
.header {
    background-color: lightblue;
    background-image: url("image.jpg");
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}

### 8. TEXT PROPERTIES:
- color: Text color (hex, rgb, named colors)
- font-family: Font type (Arial, sans-serif)
- font-size: Text size (px, em, rem, %)
- font-weight: Boldness (normal, bold, 100-900)
- text-align: Alignment (left, center, right, justify)
- text-decoration: Underline/overline (underline, none)
- line-height: Space between lines

Text Examples:
.heading {
    color: #333333;
    font-family: 'Arial', sans-serif;
    font-size: 24px;
    font-weight: bold;
    text-align: center;
    text-decoration: underline;
    line-height: 1.5;
}

### 9. BOX-SHADOW (DROP SHADOWS):
box-shadow: horizontal-offset vertical-offset blur-radius spread-radius color;

Examples:
.shadow1 {box-shadow: 5px 5px 10px gray;}
.shadow2 {box-shadow: 0 0 20px rgba(0,0,0,0.3);}
.shadow3 {box-shadow: inset 0 0 10px black;} /* Inner shadow */

### 10. WIDTH/HEIGHT UNITS:
- px: Fixed pixels (absolute)
- %: Percentage of parent element
- em: Relative to font-size of element
- rem: Relative to root element font-size
- vw/vh: Percentage of viewport width/height

Unit Examples:
.container {width: 80%;} /* 80% of parent */
.box {width: 300px;} /* Fixed 300 pixels */
.text {font-size: 1.5em;} /* 1.5 times current font size */
.full-height {height: 100vh;} /* Full viewport height */

3. Class selectors (.class), attribute selectors, pseudo-classes
