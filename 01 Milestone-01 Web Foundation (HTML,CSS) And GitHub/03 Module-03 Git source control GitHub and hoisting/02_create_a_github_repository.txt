# GITHUB REPOSITORY CREATION & EXPLORATION GUIDE

## STEP 1: CREATE GITHUB ACCOUNT
1. Go to github.com
2. Click "Sign up"
3. Enter email, password, username
4. Verify email
5. Complete setup

2. Select "New repository"
